# Findings: 妖币涨跌特征研究 (100轮完成)

## 研究概览

经过100轮系统性研究，分析了涨幅从150%到14000%+的各类妖币，涵盖:
- AI概念币 (VIRTUAL, AIXBT, AI16Z, ACT, GOAT等)
- Meme币 (PEPE, WIF, BONK, FLOKI, PNUT等)
- 老币复活 (LOOM, GAS, REEF, FUN, OG等)
- 极短周期爆发 (HIFI, ALPACA, XPIN, CUDIS, MMT等)
- 新币上市 (DOGS, HMSTR, PENGU等)

---

## 一、入场信号 (可量化)

### 1.1 横截面排名变化 (核心发现)

| 信号 | 描述 | 出现频率 | 可靠度 |
|------|------|----------|--------|
| 排名下滑蓄势 | 爆发前24h排名从前50降至后50% | 高 (75%+) | 极高 |
| 排名快速跃升 | 启动首小时排名进入Top20 | 高 (85%+) | 高 |
| 底部排名稳定 | 连续3天排名在后30%且波动<5名 | 中 (50%) | 中 |

**核心规律**: 排名变化比绝对价格更可靠。爆发前常见"排名快速下滑"现象，是主力洗盘完成的信号。

### 1.2 量价特征

| 信号 | 描述 | 出现频率 |
|------|------|----------|
| 成交量放大 | 2-3倍以上放量 | 高 (90%+) |
| 突破关键位 | 突破前高/整理区间 | 高 (85%+) |
| 成交量突变 | 成交量是前期均值的20-50倍 | 高 (80%+) |
| 底部形态 | 双底/缩量盘整后放量 | 中 (60%) |
| 长下影线阳线 | 下影线>30%，反转信号 | 中 (50%) |
| RSI超卖反弹 | RSI<35后反弹 | 中 (45%) |

### 1.3 洗盘后企稳

| 特征 | 数值范围 | 说明 |
|------|----------|------|
| 洗盘幅度 | -15%~-25% | 一次性大幅下跌 |
| 企稳时间 | 1-3天 | 成交量萎缩 |
| 启动前振幅 | <2% | 低波动蓄势 |

## 二、主升浪特征

### 2.1 启动节奏分类

| 类型 | 首小时涨幅 | 总周期 | 案例 |
|------|------------|--------|------|
| 闪电型 | >100% | 1-3天 | AVA(132%), HIFI(1147%), CUDIS(1341%), MMT(4375%) |
| 脉冲型 | 30-50% | 5-15天 | PEPE, WIF, PNUT, ALPACA |
| 长周期 | <20% | 30-90天 | ORDI, OSMO, TOKEN, FOLKS |

### 2.2 涨幅分布规律

**首日vs总涨幅**:
- 闪电型: 首日完成60-80%涨幅
- 脉冲型: 首日完成30-50%涨幅
- 长周期: 首日完成<15%涨幅

### 2.3 多波结构

多数妖币呈现2-3波上涨结构:
- **第一波**: 主力建仓+试盘
- **回调**: 洗盘，幅度15-30%
- **第二波**: 主升浪，通常最大
- **第三波(可选)**: 诱多，成交量背离

## 三、见顶信号 (可量化)

### 3.1 K线形态

| 信号 | 描述 | 可靠度 |
|------|------|--------|
| 长上影阴线 | 上影线>15%，高位出现 | 极高 |
| 天量滞涨 | 成交量新高但涨幅<2% | 极高 |
| 十字星/流星线 | 高位出现 | 高 |
| 量价背离 | 价格新高但量萎缩 | 高 |
| RSI极值 | RSI>85持续4h+ | 高 |
| 双顶形态 | 二次冲顶不破前高 | 中 |
| Bias极值 | Bias>20% | 中 |

### 3.2 成交量特征

| 指标 | 见顶信号 |
|------|----------|
| 成交量倍数 | 见顶当日成交量是前24h均值的5-10倍 |
| 量价背离 | 价格新高但成交量未新高 |
| 天量滞涨 | 成交量峰值出现在非价格峰值日 |

### 3.3 横截面特征

- 见顶时通常处于涨幅榜Top5
- 见顶后24h内排名快速下滑
- 连续2天排名下滑是确认信号

### 3.4 退场策略

| 策略 | 时机 | 预期损失 |
|------|------|---------|
| 激进 | 天量长上影当日 | <10% |
| 稳健 | 次日低开确认 | 10-15% |
| 保守 | 跌破关键支撑 | 20-30% |

## 四、可量化因子

### 4.1 入场因子 (高优先级)

```
入场信号 =
  (24h排名下滑 > 50%) AND
  (前期洗盘 > -15%) AND
  (企稳天数 >= 2) AND
  (当前振幅 < 3%)
```

```
成交量突变因子 = 当前成交量 / MA5_成交量
启动信号 = 成交量突变 > 3
```

### 4.2 见顶因子 (高优先级)

```
天量滞涨 = (成交量 > MA20_成交量 * 2) AND (|涨跌幅| < 3%)
```
来源: VIRTUAL, ETHFI, AEVO, PORTAL, COAI 等多个案例

```
上影线比例 = (最高价 - 收盘价) / (最高价 - 最低价)
见顶信号 = 上影线比例 > 0.6
```
来源: JASMY, PIXEL, DYM, STRK, ALPACA 等

```
见顶信号 =
  (上影线 > 15%) OR
  (成交量倍数 > 5x AND 涨幅 < 0) OR
  (连续2根阴线 AND 排名下滑)
```

### 4.3 动量因子改进

```
改进动量 = 原动量 * exp(-距新高时间/衰减常数)
```
来源: VIRTUAL #1

### 4.4 新币周期因子

```
新币阶段 = 上市后天数
高危期 = 上市后 5-15 天 (主升浪见顶窗口)
```
来源: 所有新币案例

## 七、100轮研究核心结论

### 7.1 核心规律

1. **排名变化是最可靠的信号**: 爆发前排名下滑，爆发后排名跃升
2. **成交量是领先指标**: 成交量见顶早于价格见顶
3. **上影线是最直观的见顶信号**: 长上影阴线后大概率开始下跌
4. **多波结构是常态**: 一波流较少，通常有2-3波

### 7.2 时间规律

1. **凌晨变盘**: 02:00-06:00是重要变盘时段
2. **周末风险**: 新币周末持仓需谨慎
3. **美盘时段**: 拉升概率较高

### 7.3 策略建议

**入场**:
- 关注排名快速下滑后企稳的币种
- 等待成交量放大确认启动
- 首小时涨幅>30%可跟进

**持仓**:
- 保持在涨幅榜前10时持有
- 设置移动止盈

**离场**:
- 出现长上影阴线立即减仓
- 成交量创新高但价格未突破时警惕
- 排名跌出前20考虑清仓

### 7.4 持仓管理

| 阶段 | 动作 | 条件 |
|------|------|------|
| 启动 | 买入 | 首小时涨幅>30%且排名进入Top20 |
| 主升 | 持有 | 保持涨幅榜前10 |
| 见顶 | 减仓 | 出现长上影或放量滞涨 |
| 确认 | 清仓 | 排名跌出前50 |
