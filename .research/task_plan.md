# Task Plan: 第51-100轮妖币特征深度研究

## Goal
完成50轮真正妖币特征研究，寻找可量化的横截面显著特征，为后续因子开发奠定基础

## 妖币定义标准

基于 MYX (+38311%) 和 RIVER (+1831%) 案例：

| 标准 | 阈值 | 说明 |
|------|------|------|
| 最小涨幅 | >500% | 至少5倍 |
| 周期特征 | <100天 | 快速爆发 |
| 关键特质 | "妖性" | 不仅仅是新币首发，要有多轮行情 |

## 研究重点

不再仅仅关注"量价齐升"，需要探索：
1. **横截面显著特征**：相对其他币种的独特性
2. **入场信号**：启动前的可量化前兆
3. **出场信号**：见顶前的危险信号
4. **时序特征**：主升浪的节奏和形态

## 研究模板

每轮研究需要分析：

### 1. 基础数据
- 涨幅时间段的K线数据
- 成交量序列
- 与BTC的相关性

### 2. 入场信号探索
- 启动前的量价特征
- 横截面排名变化
- 是否有"潜伏"迹象

### 3. 主升浪特征
- 上涨节奏（连续阳线、脉冲式）
- 成交量模式
- 回撤特征

### 4. 出场信号探索
- 见顶前的危险信号
- 天量滞涨、长上影线
- 动量因子变化

### 5. 横截面特征
- 相对BTC/ETH的alpha
- 相对同板块的表现
- 市值排名变化
