# ETH CTA 研究项目计划

## 目标
通过50轮系统性研究，发现ETH的CTA交易特征，寻找可靠的做多/做空/平多/平空离散信号，构建逻辑过硬而非过拟合的交易策略。

## 研究原则
1. **不对称性**: 多空信号分开研究，做多和做空的逻辑大概率不同
2. **逻辑优先**: 所有信号必须有合理的市场逻辑解释，拒绝纯数据挖掘
3. **单币深耕**: 专注ETH一个币种，深度挖掘其独有特征
4. **工具丰富**: 可使用通道类因子、均线、RSI等所有private/factors中的指标

## 研究方法论

每轮研究应包含:
1. **假设提出**: 基于市场逻辑的假设
2. **数据分析**: 使用ETH历史数据验证
3. **信号构建**: 设计具体的信号逻辑
4. **回测验证**: 使用回测引擎验证效果
5. **结论记录**: 记录有效/无效的发现
